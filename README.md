# 🚴 TourFrance Android

TourFrance es una aplicación Android para la gestión y consulta de información relacionada con el **Tour de Francia**. Permite al usuario consultar ciclistas, equipos, etapas y puertos de montaña, interactuar con una API REST propia y gestionar una lista de ciclistas favoritos de forma local.

La API REST a la que se conecta es la realizada en la asignatura de **Acceso a Datos** del curso pasado.

---

##  Características principales

###  Ciclistas
- Listado con nombre, especialidad, lugar de nacimiento y títulos
- Añadir, editar y borrar ciclistas contra la API

###  Equipos
- Listado con nombre, país y email
- Añadir, editar y borrar equipos contra la API

###  Etapas
- Listado de etapas del Tour con distancia, tipo y desnivel

###  Puertos de montaña
- Listado con categoría, altitud y pendiente
- Mapa interactivo con **Mapbox** que muestra la ubicación de los puertos

###  Favoritos
- Gestión local de ciclistas favoritos con notas personales
- Base de datos local con **Room**

###  Preferencias
- Tema claro/oscuro
- Unidades de distancia e idioma *(pendientes de desarrollo)*

###  Idiomas
- Español 🇪🇸
- English 🇬🇧
